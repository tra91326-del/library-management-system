# Library Management System

A Java-based console application for managing a library system using Object-Oriented Programming (OOP) and MVC architecture.

## Features

- Add new books
- View all books
- Search book by ID
- Update book information
- Delete book (in progress)

## Tech Stack

- Java
- Object-Oriented Programming (OOP)
- MVC Architecture
- ArrayList for data storage

## Status

This project is currently under development. More features will be added soon.
